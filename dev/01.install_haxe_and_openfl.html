Intro


We love coffee, pizza and traveling. So, if you prefer to write a tons of code instead of enjoying your life â€“ than you should leave this guide.

The main idea behind of our work is to pass full development flow in short period. We will start from small introduction to technologies and after that you can check our 'Mobile App Development' and 'Game Development' guides.

We will use Haxe. It's an open-source high level language with huge portion of syntax sugar. As media back-end we will use Lime and OpenFL. It's an open-source flash-like API layer based on OpenGL. For speed up our development flow, adding some cool platform specific features and complex logic we will use AppCake API and Bake build system.

1

You can download Automatic Installer for MacOS X, Windows and Linux flom Haxe download page.

You can build Haxe from sources, but we don't recomend you to do this until you need it.


2

OpenFL require Lime and Lime tools packages. It is simple to install Lime:

haxelib install lime
haxelib run lime setup

3

Now we need OpenFL and OpenFL-Native libs:

lime install openfl
lime install openfl-native

4

Almost done! We need to setup SDK for end-user targets. You don't need install them all. Install only platforms that you planing to use:

lime setup android
lime setup blackberry
lime setup tizen
lime setup windows

Note: for iOS and MacOS X you need to install XCode and should have Apple Developer Accout


5

That's it! Make a cup of coffee and let's dive deep!
